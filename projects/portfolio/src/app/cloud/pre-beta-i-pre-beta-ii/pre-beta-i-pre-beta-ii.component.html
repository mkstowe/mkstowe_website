Checkpoint System / Return to Hub

I implemented a checkpoint system so that the player does not have to completely restart a level upon death. Each time they use one of the moving platforms to get to a new floor of a dungeon, they unlock a checkpoint that will later be available in a UI selection for them to revisit later on from the hub.

Additionally, I set it up so that the player can choose to return to the hub at any time during a dungeon. This way they can easily go back if they realize they needed to get something else, or if they accidentally enter an area that they have already cleared.
